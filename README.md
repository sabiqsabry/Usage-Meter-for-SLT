# Usage Meter for SLT

Easily track your SLT broadband usage on iOS and macOS.

## About the App

Usage Meter for SLT helps you keep track of your SLT home broadband data usage in (near) real-time. With a simple and beautiful interface, you can check your total usage, and remaining data at a glance. Available on iPhone, iPad, and Mac.

**Available on the App Store**

## Features

- View multiple connections under the same account
- Home Screen Widgets
- Dark Mode support
- Native iOS and macOS design
